== Overview

This page contains all of the APIs for managing the URL filter.

[NOTE.since]
====
Available since `3.25.0`
====

[NOTE]
====
The following APIs only operate on the active filter state. The active filter state includes all dictionary and blacklist
entries excluding pending modifications. Use the management interface to manage pending modifications. There is no approval
process for the APIs and everything will be immediately added to the active filter state.
====
