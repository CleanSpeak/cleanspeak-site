Inversoft's Passport Service Now Available on IBM Cloud

Developers using IBM's cloud platform can now seamlessly add authentication, security and user management into their application design with Inversoft's Passport API

DENVER, June 6, 2017 IBM Business Partner Inversoft has made its Passport API service available on IBM Bluemix, IBM's cloud platform, to bring powerful authentication, authorization and user management to developers. Developers can now build with Passport on Bluemix and easily integrate the cloud-based identity platform into their web and mobile applications, allowing them to quickly and easily authenticate and authorize users.

The service, Passport by Inversoft, provides user login and registration for applications, saving developers the time and effort typically required to build and configure authentication and authorization properly. Additionally, Passport's responsive user interface allows developers to manage end-user data, creating visibility through custom user data, search, segmentation, localization and reporting.

Having the Passport service available as a gateway through IBM Cloud allows developers to programmatically connect Passport to their apps built on IBM Cloud, and use them alongside IBM's broad portfolio of products, including more than 150 IBM Cloud APIs and services, which span categories of cognitive intelligence, analytics, blockchain and the Internet of Things.

IBM's cloud platform, is built on open technology, giving developers the cloud power they need to build without the need to configure infrastructure. IBM Cloud has rapidly grown to become one of the largest open, public cloud deployments in the world.

Inversoft Passport is also listed in the Global Solutions Directory, an online directory containing applications, tools and services from IBM and IBM Business Partners. Passport is IBM Verified and Cloud Ready.

"Access to Passport helps remove the complexity of building and maintaining boilerplate user management infrastructure so developers can focus on innovating their own product and core business functionality", said Brian Pontarelli, Founder, CEO, Inversoft. "We're thrilled to be a part of the growing IBM Cloud community and proud to be a verified IBM Business Partner. We look forward to helping IBM clients continue to use our product."

To learn more about leveraging Passport in IBM Bluemix and to get started, visit IBM Bluemix.
